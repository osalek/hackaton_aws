Dotdeb-list
===========

This role helps to add dotdeb repository list.

Requirements
------------

This role requires ansible 1.4 or higher.

Role Variables
--------------

None

Dependencies
------------

None

Example Playbook
----------------

    - hosts: servers
      roles:
        - { role: dotdeb-list, tags: dotdeb-list }
