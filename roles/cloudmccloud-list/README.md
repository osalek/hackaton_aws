Cloudmccloud-list
=================

This role helps to add cloudmccloud repository list.

Requirements
------------

This role requires ansible 1.4 or higher.

Role Variables
--------------

None

Dependencies
------------

None

Example Playbook
----------------

    - hosts: servers
      roles:
        - { role: cloudmccloud-list, tags: cloudmccloud-list }
